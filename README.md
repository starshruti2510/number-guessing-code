<h1>Number Guessing Game</h1>
This game is a simple JAVA console game where the user has to guess the number which is selected by the computer.

The source code of the game is in the "src" folder.

-----

<h3>Features:-</h3>
  1.Random Number will be chosen by the computer.<br>
  2.User will guess the number and give the input.<br>
  3.It will give hints if the user cannot guess the number.<br>
  4.It is simple and beginner-friendly logic & code.<br>

-----

<h3>Technologies Used:-</h3>
  1.Basic JAVA<br>
  2.Git<br>

-----

<h3>How to Run:-</h3>
  1.Clone the repository.<br>
  2.Open the project folder on your IDE.<br>
  3.Compile the JAVA file.<br>
  4.Run the main class to start the program.<br>
