#Number Guessing Game
This game is a simple JAVA console game where the user has to guess the number which is selected by the computer.

The source code of the game is in the "src" folder.

-----

##Features:-
- Random Number will be chosen by the computer.
- User will guess the number and give the input.
- It will give hints if the user cannot guess the number.
- It is simple and beginner-friendly logic & code.

-----

##Technologies Used
- Basic JAVA
- Git

-----

##How to Run
- Clone the repository.
- Open the project folder on your IDE.
- Compile the JAVA file.
- Run the main class to start the program.
